# china-wcpick-pages

## 项目简介

本仓库用于归档和发布一系列独立的 HTML 页面。这些页面以静态文件的形式存放，不依赖特定后端服务，可直接通过浏览器访问。

仓库不针对任何具体域名或网站，仅作为通用 HTML 页面集合的存储与展示空间。

## 目录结构

仓库根目录下存放各 HTML 文件及相关资源。

```
/
├── index.html          # 首页（可选）
├── page1.html
├── page2.html
│   ...
└── assets/             # 公共资源文件夹（图片、样式、脚本等）
    ├── css/
    ├── js/
    └── images/
```

各页面之间相互独立，可单独引用资源文件。

## 页面归档说明

- 所有页面均为纯静态 HTML，不含动态服务端逻辑。
- 页面内容可能随时间更新，历史版本可通过 Git 提交记录追溯。
- 资源文件尽量内联或使用相对路径，以保持可移植性。
- 不保证所有页面在所有浏览器中完全兼容，但尽量遵循 Web 标准。

## 维护说明

- 欢迎提交 Issue 报告页面显示异常或内容错误。
- Pull Request 用于添加新页面或修改现有页面。
